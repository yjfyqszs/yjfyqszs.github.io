配置 Github
注册 Github 账号，新建 New repository，命名为 http://用户名.github.io

安装 Git，运行 Git Bash，设置用户名和邮箱

生成 ssh 密钥文件

在 C:/用户/你的用户名/.ssh 中会生成两个文件 id_rsa 和 id_rsa.pub ，用记事本打开 id_rsa.pub ，将内容复制到 Github 新建的 SSH key。

Hexo 配置
安装 Node.js，随后在 cmd 中运行 npm install -g hexo-cli ，即可安装 Hexo。

在 Git bash 中运行 hexo init MyHexo 初始化博客。

在 MyHexo 文件夹运行以下命令

hexo 模板中可以挑选主题模板，按说明配置

npm list --depth 0 可查看已经安装的包

部署到 Github
安装插件 npm install hexo-deployer-git --save
修改 _config.yml

Git Bash 运行以下命令

解决本地预览图片不显示问题
打开 MyHexo 文件夹的 _config.yml 文件，将 post_asset_folder 的值修改为 true

在 Git bash 中运行 npm install hexo-asset-image --save

此时新建文章后， source/_posts/ 中会生成与 md 文件同名的文件夹，将图片放在该文件夹即可

如果仍然不显示图片，根据错误提示修改 /node_modules/hexo-asset-image 里的 index.js 文件。

给文章添加密码
安装插件 npm install --save hexo-blog-encrypt
添加密码时在文章信息头部添加 password 即可
