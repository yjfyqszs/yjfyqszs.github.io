在 UE 纯蓝图工程中添加 C++ 类时，会出现编译错误，测试多种解决方法未果。

新建 UE C++ 工程，在 C++ 工程中新建插件，将编译后的插件拷贝至蓝图工程，即可在蓝图工程中使用 C++ 编译的模块。

新建 UE C++ 工程，点击 Settings-Plugins-New plugins ，UE 提供的默认插件如图
